蔚藍天空 | 空中地圖

專門為戶外飛機觀測、即時拍攝與航空愛好者設計的行動端 App 嵌入式實時空中雷達地圖系統。

🚀 專案研發： 陳麒安 CooperChen
📖 ABOUT THE PROJECT

專案簡介

本系統徹底打破了傳統 H5 網頁在 Android WebView 沙盒環境下的諸多語音死鎖、自動播放攔截與定位卡頓等硬體屏障，完美融合 Flightradar24 (FR24) 的經典飛機觀測美學，並搭載了獨創的 AndroidBridge 原生語音橋接系統與雲端資料庫同步功能。讓觀測者在機場跑道旁、航道下方，都能獲得秒級精準的航空數據與客製化的廣播體驗。

🌟 CORE FEATURES

四大核心特色

📱

1. 專為 Android WebView 設計的行動級 UI 布局

  • 底部導航列 (Bottom Navigation Bar)：針對手機與 Android 內嵌窄螢幕優化，將雷達地圖、地圖測站設定、語音客製、系統日誌完全分頁化。
  • 100% 畫面適應（No Scrollbar）：地圖全螢幕延伸，徹底解決了網頁元件在小螢幕上因橫寬不足導致折行擠壓與被迫滑動的痛點。
  • 可摺疊左側抽屜與懸浮細節卡片：點擊地圖飛機即時滑出 FR24 經典儀表板，航班列表可一秒收合，解鎖最開闊的地圖視野。
🗣️

2. 三級智能語音防護與「自訂朗讀順序」

  • Android 原生橋接 (AndroidBridge)：H5 網頁能直接呼叫 Android 原生 TextToSpeech 引擎，速度最快、發音最自然，且完全不消耗網頁內存與流量。
  • 首創語音自訂排序 (TTS Reorder)：提供視覺化調整面板，使用者可自訂前置詞、航空公司、呼號、高度、速度、升降率等物理指標的口說播報順序。
  • 5秒硬體看門狗 (Watchdog) 守護：防止 WebView 語音引擎併發死鎖而突然不讀，自動維持播報序列順暢。
🌐

3. 100% 全真實航空 ADS-B 數據直連

  • 無偽造模擬數據：對接全球真實 ADS-B（Airplanes.live 與 OpenSky Network）二次監視雷達天線數據。
  • APP 直連優先策略：在 Android 環境下優先透過 App 的本地網絡直連數據源，繞過容易斷線、速度慢的第三方 CORS 代理伺服器。
  • 漸層高度軌跡著色：飛機在天空中飛行的軌跡線會依照其飛行高度自動著色（低高度橘紅、中高度軌跡、高空巡航湛藍），完全對齊專業 FR24 邏輯。
☁️

4. 雲端資料庫預設同步 (Cloud Database Sync)

  • 自動登入備份：與雲端資料庫進行無縫連線，使用者自訂的「觀測半徑」、「語音語速」、「廣播前置詞」及「朗讀順序」會即時備份至雲端。
  • 零阻礙離線容錯：無網路時自動切換為本地離線模式，即使在深山或收訊不佳的機場旁，依然不影響 GPS 定位與觀測地圖的運行。
🛠️ TECHNICAL STACK

技術架構

前端介面
HTML5 / Tailwind CSS

日間高對比、簡約清新視覺設計。

地圖渲染
Leaflet.js Map

支援 POSITRON 淺色圖、街道圖及衛星圖快速熱切換。

後端與雲端
Firebase Ecosystem

Firebase Auth、Firestore 用於用戶自訂偏好遠端同步。

原生 App 橋接
JavascriptInterface

對接 AndroidBridge 原生 TTS，搭載 GPS watchPosition 背景監聽。

🎨
DESIGN AESTHETIC

日間高對比清新設計美學

不同於普通雷達的黑夜風格，「蔚藍天空 | 空中地圖」專為日間戶外強光設計。我們採用冰川白與淺灰色背景，在刺眼日照下依舊擁有極高可讀性。地圖上的黃金、橘紅高對比飛機圖標特別配上白色高亮描邊與深色投影，即使在複雜的衛星底圖、森林、甚至海洋上方也絕對不會「視覺失鎖」，是飛機攝影愛好者（Plane Spotters）在跑道頭追飛機、即時捕捉航班的最佳利器。
